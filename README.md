# TaskFlow - To-Do List App 
 
Aplicacion de tareas desarrollada con Ionic + Angular y Firebase. 
 
- Ionic 7 + Angular 17 
- Firebase (Auth, Firestore, Remote Config) 
- Cordova 
 
## Funcionalidades 
- Registro e inicio de sesion 
- Crear, completar y eliminar tareas 
- Categorias: Personal, Trabajo, Estudio, Hogar 
- Filtrar por categoria 
- Perfil de usuario 
- Remote Config (feature flag)
